##### SNAKE v1.0

by <*Pingu*>





###### Description

--------------

Classic Snake game with multiple difficulties, highscores,

sound effects, and pause functionality.





###### Controls

----------

Arrow Keys  - Move snake

S           - Start game

P           - Pause / Resume

M           - Toggle sound

H           - View highscores

B           - Back from highscores

R           - Restart (in game)

X           - Reset highscores (title screen only)

D           - Change difficulty (title screen)

Q           - Quit game





###### DIFFICULTIES

---------------

Easy    – Slow start, gentle speed increase

Normal  – Classic Snake feel

Hard    – Fast and challenging

Insane  – No speed Limit





###### Important notes

-------------------

* &nbsp;The game will automatically generate a file called "Highscore.txt"
* &nbsp;Highscores are stored locally in "highscore.txt"
* &nbsp;Deleting this file resets all highscores
* &nbsp;Sound requires Windows



Enjoy!
